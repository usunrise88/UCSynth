# GUI-контроллер (этап 2)

Десктоп-пульт `app/` — управляет синтом по тому же бинарному протоколу, что и `serialtest.py`, но
мышью и клавиатурой. **Стек:** чистый Go + [Gio](https://gioui.org) (без cgo на Windows). **Таргет:**
Windows; собирается и на Linux (кросс-компиляция).

## Что в v1 (ядро-контроллер)
- **Подключение** по COM-порту; авто-подсветка нативного USB-порта S3 (Espressif VID 303A) — не мост CH343.
- **Блоки параметров** строятся из дампа реестра (`LIST`) — слайдеры (float), степперы (int),
  сегменты (enum), тумблеры (bool). Двусторонняя синхра: крутишь — слышно; клампы с устройства отражаются.
  Новый параметр в прошивке появляется сам (неизвестные — в блок «Прочее»).
- **Экранная клавиатура:** мышь + ПК-клавиши (A W S E D F T G Y H U J K = C..C), октавы `Z`/`X`, кнопка **Panic**.
- **Метрики** в реальном времени: CPU / underruns / heap / uptime (из `STAT`, ~2 Гц).
- **Баннер тест-тона:** `test_tone` включён по умолчанию и перебивает ноты — баннер это показывает и даёт выключить.

Дальше по слоям: патчи (JSON, дерево, импорт/экспорт), MIDI-вход, piano-roll, графики истории STAT.

## Собрать
Go ≥ 1.25 (`app/go.mod`). Из корня репозитория:

```bash
make app-build     # → app/build/ucsynth-controller.exe (Windows, кросс из Linux)
```

Нативно на Windows:

```powershell
cd app
go build -o ucsynth-controller.exe ./cmd/controller
```

## Запустить (Windows, с железом)
1. Прошей плату (см. `docs/build-flash.md`).
2. Запусти `ucsynth-controller.exe`.
3. Нажми **Порты**, выбери порт S3 (подсвечен «UCSynth / Espressif»), **Подключить**.
4. Блоки заполнятся из реестра. Баннер попросит выключить тест-тон — выключи, играй с клавиатуры.

**Грабли:**
- Портов может быть два (нативный USB-JTAG и мост CH343). Протокол — на **нативном** (Espressif VID).
- Открытие порта держит DTR/RTS low, чтобы **не ресетить** S3 (как `serialtest.py`). Если плата всё же
  ресетится на подключении — сообщи.
- Тест-тон по умолчанию ВКЛ (перебивает ноты) — выключи через баннер.

## Разработка / проверка (без железа и дисплея)
```bash
make app-test      # чистые пакеты (go test) + кросс-vet/сборка Windows (cgo-free)
```
Gio-UI в облаке не запускается (headless, нужен Windows-бинарь) — проверяется **кросс-сборкой**;
логика (протокол/модель) — нативными тестами (в т.ч. `device` против fake-прошивки через `net.Pipe`).

## Структура `app/`
| Пакет | Роль | Тест |
|-------|------|------|
| `proto/` | кадры, CRC-16/CCITT-FALSE, кодек сообщений | golden-векторы (0x29B1), resync |
| `serial/` | перечень портов, open, DTR/RTS (go.bug.st/serial) | — (I/O) |
| `device/` | клиентский реестр из LIST, приоритетный writer (ноты > SET > STAT), throttle, echo | fake-прошивка (net.Pipe) |
| `layout/` | карта параметр→блок + подписи enum/единицы (прошивка их не шлёт) | да |
| `ui/`, `cmd/controller/` | Gio: блоки, клавиатура, метрики, окно | кросс-сборка Windows |

Контракт протокола — `docs/serial-protocol.md`. Референс-клиент на Python — `tools/serialtest.py`.
