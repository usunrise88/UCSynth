# Железо

Работать **только с наличным**. Не предлагать докупать.

## Ядро

**ESP32-S3-N16R8** — 16 МБ flash / 8 МБ PSRAM. Dual-core LX7 240 МГц, FPU, SIMD, PCNT (аппаратная квадратура, 4 юнита), нативный USB, аппаратные touch-пины.

**Не использовать для аудио:** ESP32-C6 (нет FPU), обычный ESP32 (слабее S3).

---

## Аудиовыход: PCM5102 (×5)

Модуль GY-PCM5102 — I2S DAC с распаянным minijack. Аналоговый тракт готов.

**Подключение (5 проводов к S3):**
| Модуль | S3 |
|--------|-----|
| BCK | I2S bit clock |
| DIN | I2S data |
| LCK | I2S word select (LRCK) |
| VIN | 3.3V (свой LDO на плате) |
| GND | GND |

**⚠ Грабли — причина №1 «нет звука»:**
- **SCK → GND** (не к S3!). Модуль генерит системный клок внутренним PLL из BCK. Висящий SCK = тишина/шум.
- **XSMT → 3.3V**. На земле = вечный soft-mute = тишина.
- **FMT → GND** = стандартный I2S. FLT/DEMP — по умолчанию.

**Проверка до кода:**
- Мультиметр: XSMT ≈ 3.3V при поданном питании (не 0)
- Мультиметр: SCK звонится на GND
- Анализатор на BCK/LCK/DIN: при работающем I2S идут клоки и данные

## Расширители ввода: MCP23017 (×4, нужно 3)

CJMCU-2317. 16 GPIO на чип, I2C.

| Чип | Адрес | Назначение | Адресные пины |
|-----|-------|-----------|---------------|
| #1 | 0x20 | Матрица 4×4 + Play/Menu/Shift | A0/A1/A2 → GND |
| #2 | 0x21 | 4 осц-энкодера | A0 → VCC |
| #3 | 0x22 | 4 utility + master энкодер | A1 → VCC |

**⚠ Грабли:**
- **Убрать pull-up на двух из трёх плат.** У каждой свои подтяжки на SDA/SCL; три параллельно перегружают шину.
- **Не использовать A7/B7 под входы** — на rev.D (с ~2022) вход на GPA7/GPB7 не гарантируется.
- **Энкодеры читать через прерывания ITA/ITB** (mirror mode → 1 INT-пин на чип), не поллингом — иначе теряются шаги.
- Квадратура — по таблице переходов с дебаунсом, не по одному фронту (фантомные шаги на детентах).

**Опция:** активные энкодеры → на PCNT S3 (аппаратная квадратура, без потерь).

## Дисплей: ST7796 3.5" 320×480 IPS + CTP touch

SPI-версия, 40PIN.

**⚠ Ограничения:**
- Полный кадр 320×480×2 ≈ 300 КБ — **не хранить фреймбуфер**, рисовать напрямую / малыми спрайтами
- **Частичная перерисовка** (dirty rects), событийный FPS (10–30)
- UI на Core 1, иначе дропы звука
- Питание подсветки развязать от аудио (иначе наводка)

Библиотеки: TFT_eSPI или LVGL (полосовой рендер).

## Сенсоры

**VL53L0X (×2)** — ToF дальномер, I2C, адрес **0x29** (фиксированный, меняется программно через XSHUT).
- Continuous high-speed режим (~30 мс)
- Рабочий диапазон руки: 50–400 мм
- Узкий конус (~25°), лёгкое сглаживание
- Два сенсора (XY в воздухе) — развести адреса через XSHUT

**Touch-пины S3** — аппаратная ёмкость, читается как аналоговое значение. Площадка/пружина напрямую, без доп. чипа.

**Capacitive soil sensor (×5)** — экспериментальный proximity-модулятор через ADC. Одно скалярное значение, не позиция. Медленный, дрейфует, нужна автокалибровка базовой линии.

---

## Питание

- **MB102 (×3)** — импульсный, шумноватый → **только цифра** (S3, дисплей, MCP)
- **AMS1117-3.3 (×10)** — **отдельный LDO под аудио** (PCM5102)
- Развязка: 100 нФ керамика + 10–47 мкФ электролит у VIN
- **Star ground** — аналог и цифра в одну точку, не в петлю

## Ввод

- **EC11 энкодеры (×10)** с ручками. Детентные, с кнопкой. 2 пина на вращение + 1 на кнопку.
- **Кнопки** для матрицы 4×4 + Play/Menu/Shift
- **Диоды 1N4148 (×25)** — в матрицу для N-key rollover (аккорды)
