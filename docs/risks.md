# Риски

Читать при подходе к соответствующему этапу. **Предупреждать пользователя заранее** и предлагать, как замерить проблему до того, как она проявится.

---

| Этап | Риск | Митигация |
|------|------|-----------|
| **0** Каркас | PlatformIO не тянет нужную версию ESP-IDF для S3 | Проверить в первый день. Fallback: чистый `idf.py` + расширение Espressif |
| **1** Звук | PCM5102 молчит | SCK→GND, XSMT→3.3V. Анализатор на BCK/LCK/DIN |
| **1** Звук | PSRAM не определяется | Проверить конфиг sdkconfig, модель платы N16R8 |
| **3** Полифония | Не влезут 8 голосов по CPU | **Замерять CPU на каждом голосе.** Не тянет → снизить полифонию или упростить осцилляторы (wavetable дешевле PolyBLEP) |
| **5** Эффекты | Reverb не влезет по CPU | Строить **последним**. Не влезает → упростить (меньше гребёнок) или урезать полифонию |
| **5** Эффекты | Delay/reverb не влезут в RAM | Буферы **только в PSRAM** (стерео 1с @48к ≈ 384 КБ) |
| **5** Эффекты | Overdrive даёт алиасинг | Начать без оверсэмплинга. Мешает → ×2–4 вокруг шейпера (дорого по CPU) |
| **8** Управление | Потеря шагов энкодеров | INT (ITA/ITB) вместо поллинга. Не помогло → активные энкодеры на PCNT S3 |
| **8** Управление | I2C не «едет» с 3 MCP | Убрать pull-up на двух платах |
| **9** Дисплей | Дропы звука от перерисовки | Аудио на Core 0 / UI на Core 1, частичная перерисовка, FPS cap |
| **9** Дисплей | Не хватает RAM на фреймбуфер | Не хранить полный кадр — малые спрайты / полосовой рендер LVGL |
| **11** Монтаж | Наводка дисплея в звук | Раздельное питание (MB102 цифра / AMS1117 аудио), star ground |
| **11** Монтаж | Модуль сгорел, не выпаять | Хедеров нет. Держаться на бредборде максимально долго; модули на проводах, не впаянные в перфоборд |

## Точки замера

Замеры фиксировать в `progress.md`.

| Когда | Что мерить |
|-------|-----------|
| После I2S | Underruns (недогрузы буфера) |
| Каждый добавленный голос | CPU %, свободная RAM |
| После каждого эффекта | CPU %, PSRAM |
| После дисплея | Дропы звука при активной перерисовке |

**Правило:** если метрика приближается к лимиту — сказать пользователю до того, как упрёмся.
